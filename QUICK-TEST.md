# 🚀 快速测试指南

## 预览框尺寸修复验证

### 1️⃣ 启动应用（选一种方式）

**方式A: Python服务器**
```bash
cd src
python -m http.server 3000
```

**方式B: Node.js服务器**
```bash
cd src
npx serve . -p 3000
```

**方式C: VS Code Live Server**
- 在VS Code中打开 `src/index.html`
- 右键 → Open with Live Server

---

### 2️⃣ 访问应用

打开浏览器访问: **http://localhost:3000**

⚠️ **重要**: 首次测试请强制刷新
- Windows: `Ctrl + F5`
- Mac: `Cmd + Shift + R`

### 🐛 故障排除

#### 问题：预览框还是很小

**解决**:
1. 强制刷新页面（Ctrl+F5）
2. 清除LocalStorage:
   ```javascript
   // 浏览器控制台执行
   localStorage.clear();
   location.reload();
   ```
3. 检查文件更新:
   - `src/app.js` 第7行应为 `this.zoom = 1.0;`
   - `src/index.html` 第28-50行CSS应为新样式

#### 问题：预览框空白（无内容）

**参考**: `FIX-SOLUTION.md` - iframe Blob URL渲染修复

#### 问题：缩放按钮不工作

**检查**:
1. 打开控制台（F12）查看错误
2. 确认已创建日报并选中
3. 刷新页面重试
